# gacha-RPG

一款基于明日方舟风格的回合制抽卡RPG游戏。

## 游戏特色

- 🎰 **抽卡系统**：3-6星干员池，保底机制
- ⚔️ **回合制战斗**：AT条行动顺序，技能能量管理
- 👥 **队伍编成**：最多4人队伍，队长技能加成
- 🔮 **召唤系统**：召唤师可召唤流形参战
- 🎭 **时装系统**：干员时装切换
- 🌀 **无尽模式**：挑战极限，获取无尽币

### 🔮 召唤机制详解

#### 📖 召唤师是什么？

召唤师是一类特殊的干员，能够在战斗中召唤【召唤物】参战。召唤物是独立的战斗单位，拥有自己的行动回合，可以独立攻击敌人。

目前游戏中的召唤师：
| 召唤师 | 召唤物 | 特点 |
|:---:|:---:|:---|
| **缪尔赛思** | 流形 | 独立战斗单位，可通过技能强化 |

> 注：凯尔希的Mon3tr召唤功能尚未实装

---

#### ⚙️ 召唤机制规则

##### 1️⃣ 召唤时机
```
┌─────────────────────────────────────────────┐
│  召唤时机                                     │
├─────────────────────────────────────────────┤
│  ⭐ 首次行动：立即召唤1个召唤物               │
│  ⏰ 之后：每3回合自动召唤1个                  │
│  💀 召唤物死亡：下次行动立即补充              │
└─────────────────────────────────────────────┘
```

##### 2️⃣ 召唤位限制
- **最大召唤位**：4个（全队共享）
- **配额分配**：多个召唤师平均分配
  - 1个召唤师：4个位置
  - 2个召唤师：各2个位置

##### 3️⃣ 属性继承
召唤物 **100%继承** 召唤者的基础属性（召唤物本身没有固定属性）：

| 召唤者属性 | 召唤物属性 |
|:---:|:---:|
| HP 800 | HP **800** |
| ATK 150 | ATK **150** |
| DEF 40 | DEF **40** |
| SPD 95 | SPD **95** |

##### 4️⃣ 召唤物特性
- ✅ 拥有独立的HP和行动回合
- ✅ 按SPD加入行动顺序（AT条）
- ✅ 可以被治疗技能恢复
- ✅ 可以被召唤师技能强化
- ❌ 只能使用【普攻】
- ❌ 没有能量系统
- ❌ 召唤者死亡时一同消失

---

#### 🎯 召唤物强化系统

缪尔赛思的技能可以给召唤物叠加各种强力buff：

| 技能 | 召唤物获得效果 |
|:---:|:---|
| **渐进性润化** | ATK +20%，SPD +10（可叠加） |
| **生态耦合** | 每回合回血15%，攻击变为二连击 |
| **浅层非熵适应** | 普攻附带眩晕1回合 |

##### Buff叠加示例
```
第1轮：渐进性润化 → ATK+20%, SPD+10
第2轮：渐进性润化 → ATK+40%, SPD+20
第3轮：生态耦合   → ATK+40%, SPD+20, 二连击, 回血15%/轮
第4轮：浅层非熵适应 → ATK+40%, SPD+20, 二连击, 回血, 眩晕攻击
```

**完全体流形**（以缪尔赛思为召唤者，叠加2次渐进性润化后）：
- ATK：150 × (1+40%) = **210**
- SPD：95 + 20 = **115**（超过召唤者）
- 二连击：每次攻击2次，实际输出 **420**
- 回血：每回合恢复 800×15% = **120 HP**
- 眩晕：每次攻击附带1回合眩晕

---

#### 🎮 召唤流派玩法

##### 开局节奏
```
回合1：缪尔赛思首次行动 → 自动召唤流形
       使用【渐进性润化】→ 全队+15能量，自身+流形ATK/SPD提升

回合2：流形行动 → 普攻敌人
       缪尔赛思行动 → 继续【渐进性润化】叠buff

回合3：使用【生态耦合】→ 流形获得二连击+回血

回合4+：流形每回合双倍输出 + 持续回血
        缪尔赛思继续充能，择机【浅层非熵适应】
```

##### 队伍搭配建议
| 位置 | 干员 | 作用 |
|:---:|:---:|:---|
| 队长 | 缪尔赛思 | 召唤物+10%ATK加成 |
| 输出 | 陈/银灰 | 快速清场 |
| 暗杀 | 红 | 处决高价值目标 |
| 治疗 | 凯尔希/白面鸮 | 保护召唤者 |

##### 进阶技巧
1. **保护召唤者**：缪尔赛思死亡=流形消失，优先保护
2. **合理充能**：渐进性润化给全队充能，加速其他干员技能循环
3. **二连击优先**：生态耦合的二连击是流形输出翻倍的关键
4. **眩晕控制**：浅层非熵适应的眩晕可以打断敌人行动

#### 💡 召唤机制的战略价值

1. **额外行动次数**：流形相当于第5个队员，增加整体行动频率
2. **分散火力**：敌人可能攻击召唤物，减轻主力压力
3. **持续输出**：二连击后流形输出可观，且无需能量
4. **控制能力**：眩晕攻击可以打断敌人关键行动
5. **能量辅助**：三个技能都给全队充能，加速整体节奏

## 🎮 游戏玩法

### 基础操作
1. **抽卡**：消耗抽卡券获取干员
2. **编队**：选择最多4名干员组队
3. **战斗**：选择关卡挑战
4. **时装**：为干员更换外观

### 战斗系统
- **AT条**：按SPD决定行动顺序
- **能量**：普攻获得30，技能消耗不同
- **眩晕**：跳过行动回合
- **护盾**：敌人护盾需要击破
