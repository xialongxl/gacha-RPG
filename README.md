## 📖 项目介绍

**明日方舟：永恒回想录** 是一款基于纯前端技术开发的回合制 RPG 游戏，灵感来源于《明日方舟》。游戏包含完整的抽卡系统、回合制战斗、无尽模式挑战以及 SmartAI 深度学习系统等丰富玩法。

### ✨ 亮点

- 🎰 **抽卡系统** - 完整的概率机制与保底系统
- ⚔️ **回合制战斗** - SPD 行动条、破盾机制、技能连招
- 🔮 **召唤系统** - 独特的召唤师职业与流形召唤物
- 🏰 **无尽模式** - 词缀系统、Roguelike 强化、层层递进的挑战
- 🧠 **SmartAI** - 基于 IndexedDB 的深度学习敌人 AI
- 🎭 **Spine 动画** - 精美的干员 Spine 2D 动画
- 💾 **本地存档** - 支持多存档管理与导入导出

## 🚀 快速开始

### 在线试玩

> 🔗 [GitHub Pages 在线试玩](https://xialongxl.github.io/gacha-RPG/)

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/xialongxl/gacha-RPG.git

# 进入项目目录
cd gacha-RPG

# 方式1：直接打开 index.html（推荐）
# 双击 index.html 即可在浏览器中运行

# 方式2：使用本地服务器
npx serve .
# 或
python -m http.server 8080
```

### 部署到 GitHub Pages

1. Fork 本仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选择 `/ (root)`
4. 保存后即可通过 `https://[用户名].github.io/gacha-RPG/` 访问

## 🎮 游戏特色

### 🎰 抽卡系统

| 星级 | 概率 | 保底 | 满潜转金币 |
|:---:|:---:|:---:|:---:|
| ⭐⭐⭐⭐⭐⭐ 6星 | 2% | 20抽保底 | 500 |
| ⭐⭐⭐⭐⭐ 5星 | 8% | - | 200 |
| ⭐⭐⭐⭐ 4星 | 50% | - | 50 |
| ⭐⭐⭐ 3星 | 40% | - | 20 |

- 📅 **每日签到**：获取免费抽卡券
- 🎫 **保底机制**：20抽内必出 6 星干员
- 💎 **潜能系统**：每级 +5% 全属性加成

### ⚔️ 回合制战斗

#### 行动条 (AT Bar)
- 基于 **SPD（速度）** 属性决定行动顺序
- SPD 越高，行动越频繁

#### 破盾机制
| 操作 | 破盾量 |
|:---|:---:|
| 普通攻击 | 1 格 |
| 破甲斩 | 2 格 |
| 重锤 | 3 格 |
| 粉碎 | 4 格 |

护盾破碎后：
- 🔻 眩晕 1 回合
- 🔻 防御归零
- 🔻 伤害翻倍

#### 能量系统
- 普攻获得 **30 能量**
- 受击获得 **20 能量**
- 技能消耗能量释放

### 🔮 召唤系统

缪尔赛思等召唤师干员可召唤"流形"作战：

| 配置项 | 数值 |
|:---|:---:|
| 召唤位总数 | 4 |
| 首次召唤数量 | 1 |
| 属性继承比例 | 100% |
| 刷新间隔 | 每 3 次行动 |

召唤物特性：
- 继承召唤者的基础属性
- 可接收召唤者的 Buff 加成
- 召唤者死亡时召唤物消失

### 🏰 无尽模式

挑战无尽层数，获取无尽币和时装券！

#### 层数奖励
- 🎖️ 每层获得 **2 无尽币**
- 🎖️ 击败 BOSS 额外 **+10 无尽币**
- 👗 **100 无尽币** = 1 时装券

#### 精英与 BOSS
| 类型 | 出现层数 | 额外词缀 | 保底词缀 |
|:---|:---:|:---:|:---|
| 精英 | 每 5 层 | +1 | 稀有+ |
| BOSS | 每 10 层 | +2 | 传说 |

## 🔥 词缀系统

无尽模式中敌人会随机获得词缀，增加挑战难度。

### 词缀数量规则

| 层数区间 | 词缀数量 |
|:---:|:---:|
| 1-9 层 | 0 |
| 10-19 层 | 1 |
| 20-29 层 | 2 |
| 30-39 层 | 3 |
| 40-49 层 | 4 |
| 50+ 层 | 5 |

### 普通词缀 (60%)

| 词缀 | 图标 | 效果 |
|:---|:---:|:---|
| 反伤 | 🦔 | 受到攻击时反弹 15% 伤害 |
| 回血 | 💚 | 每回合恢复 5% 最大生命 |
| 狂化 | 😤 | HP 低于 30% 时攻击力 +50% |
| 迅捷 | 💨 | 速度 +15 |
| 坚韧 | 🛡️ | 防御力 +25% |

### 稀有词缀 (30%)

| 词缀 | 图标 | 效果 |
|:---|:---:|:---|
| 连击 | ⚔️ | 普攻时 30% 概率攻击两次 |
| 嘲讽 | 😠 | 强制敌人优先攻击自己 |
| 护盾 | 🔰 | 首次受击伤害减少 50%（一次性） |
| 闪避 | 💫 | 20% 概率闪避攻击 |
| 吸血 | 🩸 | 造成伤害时恢复 15% |

### 传说词缀 (10%)

| 词缀 | 图标 | 效果 |
|:---|:---:|:---|
| 分裂 | 👥 | 死亡时分裂为 2 个小型单位 |
| 爆炸 | 💥 | 死亡时对所有敌人造成 30% 最大 HP 伤害 |
| 不死 | 💀 | 首次致死伤害时恢复 30% HP |
| 强化光环 | ✨ | 队友攻击力 +15% |
