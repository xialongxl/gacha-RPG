# gacha-RPG

一款基于明日方舟风格的回合制抽卡RPG游戏。

## 游戏特色

- 🎰 **抽卡系统**：3-6星角色池，保底机制
- ⚔️ **回合制战斗**：AT条行动顺序，技能能量管理
- 👥 **队伍编成**：最多4人队伍，队长技能加成
- 🔮 **召唤系统**：召唤师可召唤流形参战
- 🎭 **时装系统**：角色时装切换
- 🌀 **无尽模式**：挑战极限，获取无尽币

## 🎮 游戏玩法

### 基础操作
1. **抽卡**：消耗抽卡券获取干员
2. **编队**：选择最多4名干员组队
3. **战斗**：选择关卡挑战
4. **时装**：为干员更换外观

### 战斗系统
- **AT条**：按SPD决定行动顺序
- **能量**：普攻获得30，技能消耗不同
- **眩晕**：跳过行动回合
- **护盾**：敌人护盾需要击破
