### 🥇 艾雅法拉 - 烈焰毁灭者 【评分: 10.0】

| 属性 | 数值 | 评价 |
|:---:|:---:|:---|
| HP | 600 | ⭐⭐ 最低，需要保护 |
| ATK | **220** | ⭐⭐⭐⭐⭐ **全游最高** |
| DEF | 40 | ⭐ 脆皮 |
| SPD | 80 | ⭐⭐⭐ 中等 |

**技能系统：**
| 技能 | 消耗 | 效果 | 评价 |
|:---:|:---:|:---|:---|
| 二重咏唱 | 20 | SPD+60，第2次起额外ATK+60%（可叠加） | ⭐⭐⭐⭐⭐ 超强叠加buff |
| 点燃 | 30 | 370%单体+185%溅射，目标DEF-25%(2回合) | ⭐⭐⭐⭐⭐ 爆发+控制 |
| 火山 | 100 | 自身ATK+130%后随机攻击6个敌人 | ⭐⭐⭐⭐⭐ 终极爆发 |

**核心机制：**
1. **二重咏唱叠加**：每次使用SPD+60，第2次起ATK+60%叠加
2. **点燃溅射**：主目标370%，其他敌人185%，目标DEF-25%持续2回合
3. **火山**：先ATK+130%，再随机6次攻击，配合叠加buff伤害爆炸

**最优循环：**
```
回合1：二重咏唱 → SPD+60（加速行动）
回合2：二重咏唱 → SPD+120, ATK+60%
回合3：二重咏唱 → SPD+180, ATK+120%（已经极快）
回合4：点燃 → 370%×(1+120%)=814%单体 + 溅射 + DEF减益
回合5：火山 → ATK再+130%后6连击清场
```

**综合评价：**  
> 艾雅法拉是当之无愧的**第一输出**。二重咏唱的SPD+60和叠加ATK+60%让她越打越快越打越强。点燃的370%+溅射+持续减防提供了恐怖的爆发和控制。火山的ATK+130%后6连击，配合叠加buff可以轻松清场。唯一缺点是生存能力差，需要治疗保护。

## 🔮 召唤机制详解

### ⚙️ 召唤机制规则

#### 1️⃣ 召唤时机
```
┌─────────────────────────────────────────────┐
│  召唤时机                                     │
├─────────────────────────────────────────────┤
│  ⭐ 首次行动：立即召唤1个召唤物               │
│  ⏰ 之后：每3回合自动召唤1个                  │
│  💀 召唤物死亡：下次行动立即补充              │
└─────────────────────────────────────────────┘
```

#### 2️⃣ 召唤位限制
- **最大召唤位**：4个（全队共享）
- **配额分配**：多个召唤师平均分配

#### 3️⃣ 属性继承
召唤物 **100%继承** 召唤者的基础属性

#### 4️⃣ 召唤物特性
- ✅ 拥有独立的HP和行动回合
- ✅ 按SPD加入行动顺序（AT条）
- ✅ 可以被治疗技能恢复
- ✅ 可以被召唤师技能强化
- ❌ 只能使用【普攻】
- ❌ 没有能量系统
- ❌ 召唤者死亡时一同消失
