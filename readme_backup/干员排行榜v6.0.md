## 🔄 版本更新概览 (v5.0 → v6.0)

*   **数值重构**：引入“突破系统”，6星干员可选择**属性突破**（全属性+100%）或**速度突破**（SPD+40%），导致强度天花板大幅拉升。

*   **召唤崛起**：召唤物享受 100% 属性继承与双倍突破收益，召唤流正式跻身 T0。

## 📈 6星干员深度测评 (完全体)

### 1. 🌹 迷迭香 - 核爆天尊
> **定位**：物理输出天花板 / 余震机制核心

**【完全体面板】** (满潜+属性突破)
*   **HP**: 1000
*   **ATK**: **510** (全游第一)
*   **DEF**: 120
*   **SPD**: 95

**【核心技能】**
*   **思维膨大**：ATK +180% (可叠加)。全游最强自Buff，叠满两层后面板 ATK 可达 **2346**。
*   **末梢阻断**：余震次数+2，范围化。让单次普攻变成“主目标+溅射+多次余震”的弹幕攻击。

**【突破收益分析】**
迷迭香的伤害公式为 `(基础ATK × Buff倍率) × (1 + 余震次数 × 0.5)`。
属性突破带来的基础 ATK 翻倍 (255 -> 510)，会被后续的所有乘区放大，最终收益远超速度突破。

**【实战评价】**
需要 2-3 回合暖机，一旦启动，单回合输出可达 10000+，足以秒杀无尽 50 层的任何非 BOSS 单位。唯一的缺点是身板脆，必须绑定夜莺。

## 🎮 Roguelike 强化搭配指南

在 v6.0 版本中，Roguelike 强化选项的选择应服务于“完全体”干员的特性：

1.  **ATK 强化 (atkUp)**：⭐⭐⭐⭐⭐
    *   迷迭香和夜莺的核心属性。基础越高，百分比收益越大。

2.  **能量强化 (energyUp)**：⭐⭐⭐⭐
    *   让迷迭香快速启动 `思维膨大`，或让夜莺开局即可释放 `圣域`。

    *   对于高攻的迷迭香和艾雅法拉，10% 的吸血往往能一口回满，减轻治疗压力。

4.  **免死金牌 (extraLife)**：⭐⭐⭐⭐⭐ (高层必选)
    *   防止满血被秒杀的唯一保险。高层容错率的核心。

## ⚙️ 机制详解 (附录)

### 1. 突破机制
*   **条件**：6星干员 + 满潜能 (13级) + 10000 金币。
*   **属性突破**：HP/ATK/DEF 加成从 +60% 提升至 +100%。适合吃面板的干员。
*   **速度突破**：SPD 加成 +40%。适合控制、先锋、刺客类干员。

### 2. 行动条 (Action Gauge) 系统
v6.0 引入了全新的动态时间轴系统，替代了旧的回合制：
*   **Action Gauge (AG)**：每个单位拥有一个行动条，当 AG 累积满 10000 时获得行动回合。
*   **速度 (SPD)**：决定 AG 的累积速度。SPD 越高，跑条越快。
*   **Time To Act (TTA)**：即距离下次行动的时间。`TTA = (10000 - 当前AG) / SPD`。
*   **多动机制**：若某单位 SPD 极高（如满突破陈 SPD 140），其 TTA 将显著低于低速单位（如重装 SPD 40）。在低速单位行动一次的时间内，高速单位可能已经行动了 3-4 次。
*   **实战意义**：在 AT 条上可以看到每个单位的 TTA 倒计时。利用高 SPD 进行“插队”和“连动”是高层控场的关键。

### 3. 余震机制 (迷迭香专属)
*   每次普攻/技能后触发。
*   伤害 = 50% ATK × 余震次数。
*   `末梢阻断` 可叠加余震次数，理论上无上限，是迷迭香伤害爆炸的根源。

### 3. 召唤机制 (缪尔赛思)
*   **属性继承**：100% 继承本体面板 (含突破加成)。
*   **Buff共享**：缪尔赛思的技能 Buff 同时作用于本体和召唤物。
