4. **思维膨大**：ATK+180%是全游最强自增益，可无限叠加

**最优循环：**
```
回合1：思维膨大 → ATK+180%（基础255 → 实际459）

回合4：巨剑投射 → 高额伤害+3次余震范围攻击
回合5：末梢阻断 → 余震变为5次+攻击
```

> 迷迭香是与艾雅法拉并列的**顶级输出**。她拥有全游最高的255ATK，配合思维膨大的ATK+180%叠加，可以轻松达到恐怖的攻击力。余震机制是她的核心特色：每次攻击后额外造成多次余震伤害，通过末梢阻断可以让余震次数无限叠加。"如你所愿"的全体眩晕+DEF-50%提供了强大的控制能力。缺点是HP极低（500），是6星中最脆的，需要夜莺等治疗保护。

## 🔮 召唤机制详解

### ⚙️ 召唤机制规则

#### 1️⃣ 召唤时机
```
┌─────────────────────────────────────────────┐
│  召唤时机                                     │
├─────────────────────────────────────────────┤
│  ⭐ 首次行动：立即召唤1个召唤物               │
│  ⏰ 之后：每3回合自动召唤1个                  │
│  💀 召唤物死亡：下次行动立即补充              │
└─────────────────────────────────────────────┘
```

#### 2️⃣ 召唤位限制
- **最大召唤位**：4个（全队共享）
- **配额分配**：多个召唤师平均分配

#### 3️⃣ 属性继承
召唤物 **100%继承** 召唤者的基础属性

#### 4️⃣ 召唤物特性
- ✅ 拥有独立的HP和行动回合
- ✅ 按SPD加入行动顺序（AT条）
- ✅ 可以被治疗技能恢复
- ✅ 可以被召唤师技能强化
- ❌ 只能使用【普攻】
- ❌ 没有能量系统
- ❌ 召唤者死亡时一同消失

## ⚡ 余震机制详解

### 🌹 迷迭香的余震系统

#### 余震基础
- 每次普攻/末梢阻断后触发余震
- 余震伤害 = 50%ATK × 余震次数
- 初始余震次数：1次

#### 余震强化
```
┌─────────────────────────────────────────────┐
│  末梢阻断效果                                 │
├─────────────────────────────────────────────┤
│  📈 ATK +55%（可叠加）                       │
│  ⚡ 余震次数 +2（可叠加）                    │
│  🎯 余震范围化（攻击主目标+相邻敌人）        │
│  💫 普攻和余震均有20%概率眩晕                │
└─────────────────────────────────────────────┘
```

#### 余震伤害计算示例
```
基础ATK 255
思维膨大×2 后：255 × (1 + 180% × 2) = 918
末梢阻断×2 后：余震次数 = 1 + 2 + 2 = 5次

单次普攻伤害：
- 主伤害：918 × 100% = 918
- 余震伤害：918 × 50% × 5 = 2295（范围化后对相邻敌人同样生效）
- 总计：918 + 2295 = 3213（单目标）
```
