## 📖 项目介绍

**明日方舟：永恒回想录** 是一款基于纯前端技术开发的回合制 RPG 游戏，灵感来源于《明日方舟》。游戏包含完整的抽卡系统、回合制战斗、无尽模式挑战、SmartAI 深度学习系统以及干员突破系统等丰富玩法。

### ✨ 亮点

- 🎰 **抽卡系统** - 完整的概率机制与保底系统
- ⚔️ **回合制战斗** - SPD 行动条、破盾机制、技能连招
- 🔮 **召唤系统** - 独特的召唤师职业与流形召唤物
- 🏰 **无尽模式** - 词缀系统、Roguelike 强化、层层递进的挑战
- 🌟 **突破系统** - 6星满潜能干员可突破为7星
- 🎟️ **复活券** - 无尽模式全军覆没时复活全队
- 🔗 **接力券** - 记录撤退层数，下次从该层继续
- 🧠 **SmartAI** - 基于 IndexedDB 的深度学习敌人 AI
- 🎭 **Spine 动画** - 精美的干员 Spine 2D 动画
- 💾 **本地存档** - 支持多存档管理与导入导出

## 🚀 快速开始

### 在线试玩

> 🔗 [GitHub Pages 在线试玩](https://xialongxl.github.io/gacha-RPG/)

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/xialongxl/gacha-RPG.git

# 进入项目目录
cd gacha-RPG

# 方式1：直接打开 index.html（推荐）
# 双击 index.html 即可在浏览器中运行

# 方式2：使用本地服务器
npx serve .
# 或
python -m http.server 8080
```

### 部署到 GitHub Pages

1. Fork 本仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选择 `/ (root)`
4. 保存后即可通过 `https://[用户名].github.io/gacha-RPG/` 访问

## 🎮 游戏特色

### 🎰 抽卡系统

| 星级 | 概率 | 保底 | 满潜转金币 |
|:---:|:---:|:---:|:---:|
| ⭐⭐⭐⭐⭐⭐ 6星 | 2% | 20抽保底 | 500 |
| ⭐⭐⭐⭐⭐ 5星 | 8% | - | 200 |
| ⭐⭐⭐⭐ 4星 | 50% | - | 50 |
| ⭐⭐⭐ 3星 | 40% | - | 20 |

- 📅 **每日签到**：获取免费抽卡券
- 🎫 **保底机制**：20抽内必出 6 星干员
- 💎 **潜能系统**：每级 +5% 全属性加成（最高13级，+60%）

### ⚔️ 回合制战斗

#### 行动条 (AT Bar)
- 基于 **SPD（速度）** 属性决定行动顺序
- SPD 越高，行动越频繁

#### 破盾机制
| 操作 | 破盾量 |
|:---|:---:|
| 普通攻击 | 1 格 |
| 破甲斩 | 2 格 |
| 重锤 | 3 格 |
| 粉碎 | 4 格 |

护盾破碎后：
- 🔻 眩晕 1 回合
- 🔻 防御归零
- 🔻 伤害翻倍

#### 能量系统
- 普攻获得 **30 能量**
- 受击获得 **20 能量**
- 技能消耗能量释放

## 🌟 干员突破系统

6星干员满潜能（13级）后可进行突破，消耗 **10000 金币**，突破后显示为 **7星**。

### 突破选择（二选一）

| 选项 | 效果 | 说明 |
|:---|:---|:---|
| 💠 **属性突破** | HP/ATK/DEF 总计 +100% | 将原有的+60%补满至+100% |
| ⚡ **速度突破** | SPD +40% | 获得40%速度加成，其他属性保持+60% |

> ⚠️ 突破后无法更改，请谨慎选择！

## 🔮 召唤系统

缪尔赛思等召唤师干员可召唤"流形"作战：

| 配置项 | 数值 |
|:---|:---:|
| 召唤位总数 | 4 |
| 首次召唤数量 | 1 |
| 属性继承比例 | 100% |
| 刷新间隔 | 每 3 次行动 |

### 召唤物特性
- 继承召唤者的基础属性
- 可接收召唤者技能的 Buff 加成
- 召唤者死亡时召唤物消失

### Buff持续时间

| Buff类型 | 持续时间 | 来源技能 |
|:---|:---:|:---|
| ATK%/SPD | 永久 | 渐进性润化 |

| 二连击 | 3回合 | 生态耦合 |
| 嘲讽 | 2回合 | 生态耦合 |
| 攻击附带眩晕 | 2回合 | 浅层非熵适应 |

## 🛒 无尽币商店

使用无尽币可购买以下道具：

| 道具 | 图标 | 价格 | 效果 |
|:---|:---:|:---:|:---|
| 复活券 | 🎟️ | 1000 无尽币 | 无尽模式全军覆没时可使用，复活全队继续挑战 |
| 接力券 | 🔗 | 800 无尽币 | 战斗胜利后撤退时可记录层数，下次从该层继续 |
| 时装券 | 👗 | 1 无尽币 | 用于购买干员时装 |

### 复活券机制
- 全军覆没时弹出使用提示
- 使用后重新挑战当前层
- 全队满血满状态
- 保留累计奖励

### 接力券机制
- 战斗胜利后撤退时可选择记录层数
- 记录时**不消耗**接力券
- 下次进入无尽模式时可选择使用接力
- **使用接力时**消耗1张接力券
- 从记录层数+1开始挑战
